# 电商系统

主要包含：
- shop-admin 电商后台管理系统
- shop-server 电商后端

## 本地启动

```shell
# 安装依赖
pnpm i

# 启动电商后台管理系统
pnpm dev

# 启动电商后端
pnpm dev:server
```

启动之后访问链接：[http://localhost:3060/products](http://localhost:3060/products)
